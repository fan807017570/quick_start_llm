# 党员活动日

***\*活动意义\****：党员“不忘初心、牢记使命”主题教育。护学活动，服务学生、方便学生。明确每一个党员自己的身份所在和责任所在，通过开展活动，让党员干部直接参加了相关的服务，提高了广大党员干部的为人民群众服务的意识。

***\*活动时间\****：每大周第二小周周五下午12点00分—13点30分

***\*活动地点\****：新实大道

***\*活动人员及安排：\****

总指挥：张如银执行指挥：周剑军、刘国斌交通疏导组：严云强、郭天兴、周剑军

护学一组：李美红、刘康章、宁丽琴 

护学二组：杨卫华、孙芳玉、张慧珍  

拍照宣传组：贾丽娟、周丽婷

***\*活动具体内容：\****

1. 11点5分全体党员佩戴党徽在校门口集合。

 

1. 疏导组疏导家长接送学生车辆以及公交车。

 

1. 护学组安排学生排队、有秩序上车返家。
