# 关于2015年度全市民办高中阶段学校年检结果的通报

饶教字[2016]174号

各县（市、区）教体局、市直民办中职学校：

为进一步贯彻落实《中华人民共和国民办教育促进法》及其实施条例，根据《关于开展2015年全市民办高中阶段学校年度检查工作的通知》（饶教字[2016]57号）要求，市教育局组成检查组于2016年4月18日至4月23日，组织对全市42所民办高中阶段学校2015年度办学情况进行年度检查。检查组通过听取汇报、查阅资料、察看现场、反馈评议等形式，对受检学校的办学条件、办学行为、社会效益、资产与财务管理情况等进行量化评分，综合评定。现将年检结果通报如下：

年度检查合格和基本合格的民办学校，于2016年8月1日-12日携带学校办学许可证副本到上饶市教育局发展规划科办理年检手续，逾期不予办理。

年度检查基本合格的学校，严格按照《关于2014年度全市民办高中阶段学校依法办学情况年检结果的通报》（饶教字【2015】124号）有关要求认真加以整改。县（市、区）教育行政部门接本通知后，及时转发至本辖区内各民办高中阶段学校，并督促指导基本合格学校进行认真整改，整改期限为一年。

年检不合格学校终止其高中阶段办学资格，有关县（市、区）教育局要督促被终止高中阶段办学资格学校在收文后一个月内把市教育局原核发的学校办学许可证（正、副本）、相关印章一并上交上饶市教育局发展规划科。今后，未经市教育局批准，被终止高中阶段办学资格的学校不得从事高中阶段教育教学活动。
