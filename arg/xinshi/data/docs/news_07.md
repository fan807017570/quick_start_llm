# 上饶考生，凭高考准考证可免费乘坐公交车

在这万木葱茏的美好季节，莘莘学子十余年的努力与坚持，都将在六月绽放出明艳的花朵。2024年高考即将拉开帷幕，莘莘学子将执笔为剑，怀揣梦想，开启人生新篇章！

高考期间所有高考学生持准考证即可免费乘坐城市公交车。

为保障高考送考工作顺利开展，上饶公交开通爱心送考“金榜题名”“成功上岸”“一举高中”“状元”“前程似锦”“文曲星”等祝福专列，于中心城区投放22辆公交车、22名驾驶员、19名志愿者；广丰区投放19 辆公交车、19名驾驶员为上饶中学、上饶市一中、上饶市二中、康桥中学、恒立中学、职业高中、新实学校、丰溪学校、广丰中学等学校师生提供点对点送考服务。
