# 高二级部野外拓展活动！ —记玉山欧月庄园一日游

 秋高气爽，正是外出的好时节。为了不断推行素质教育，锻炼学生的实践能力走出课堂，体验生活，培养学生的自主能力和创新意识，增强团队的合作意识。2018年11月23日我校高二级部全体师生和部分家长开展野外拓展冬游活动！

　　 在各班主任的带领下，高二级部全体师生走进玉山欧月庄园拓展中心，穿过浪漫的欧月长廊，映入眼帘的就是滑草场，同学们激情澎湃的涌入中心，眼中散发出跃跃欲试的光芒。但在尝试之前都需先突破内心的一道防线，战胜自己的恐惧。新实的学子都是好样的，各个勇往直前，草地上惊叫声不断。
    在庄园里有着:旱地滑雪、极速滑草、军事打靶、山地卡丁车、小摆锤、田野射击，网红桥、星际飞车等各式各样的体验项目。
    有着军人梦的少男少女们在这里都得以实现—射击场和射箭场。在箭拔弩张的时刻，瞄准方向，十环的那一刻就像是造就梦想一样。
    最令人印象深刻的便是儿童手摇船和网红桥了。没错，就是“儿童手摇船”，在这个爱玩的年纪，是远远抵挡不了坐上去的冲动，虽然大多数都翻船，落水的滋味不好受，可同学们依然兴致昂扬，更加放开手脚的去体验了番网红桥的乐趣，结果同学们就是像下饺子一样一个接着一个掉进泥泞的田里，玩的不亦乐乎。一上午的活动结束后，同学们在庄园生态餐厅进行了午餐，同学们有秩序的进行着，脸上洋溢着幸福的笑容。

活动总结:一个人在事业上能取得成功，他的脚下必定有人甘当基石。一个团队能有辉煌的成绩，离不开各个成员的密切配合和团队意识。
