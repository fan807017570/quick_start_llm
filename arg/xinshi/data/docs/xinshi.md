

# 第一部分：学校基础概况（合规宣传核心模块） 

## 1.1 学校基本信息

 广丰区新实学校创办于2000年6月， 是一所民办完全中学， 位于广丰区吴村镇玉田村广管公路边。学校占地面积30204平方米（45亩），建筑面积12120平方米，绿化面积1000余平方米。拥有藏书总量30500余册。学校现有专兼职教师100多人，行政后勤工作人员30多人，专任教师学历达标，教职工中高级教师多人，党员16人，全校教师的平均年龄约为36岁。学校高度重视教师队伍建设和教育模式创新，师资力量和教学水平在同类型学校中领先，不但缔造了“三年四状元”的高考神话，而更为难能可贵的是，将一大批当年中考并不突出的学生以及在往年高考战场中暂时失利者，在素质教育的熔炉中锻造出高考优秀者、卓越者，实现了低进优出的喜人育人成果，为社会培养了一批又一批优秀人才，并连续六年获得广丰区“先进学校”荣誉称号，先后获得上饶市高考质量优胜单位和进步单位等20余项荣誉，被江西省教育厅授予“江西省高中特色示范实验学校”称号。

​    学校初高中教育两线并行，共有40个教学班，在校学生2200余人。宿舍楼、图书阅览室、微机室、语音室、多媒体室、书画室、舞蹈室等基础设施俱全，空调、热水系统等生活设施齐备。随着信息化、大数据、云计算、人工智能时代的到来，学校校园宽带局域网、校园智能广播系统和多媒体教学、“智慧课堂”设备等陆续建成投入使用，受到广大师生的普遍欢迎。

​    学校创办近二十年以来，一直以“办人民满意教育”为核心追求，坚持培养“有责任的中国人”的高远办学目标，以“做科研教师、育创新学生”为教风，以“管理立校，质量强校,品牌兴校”为治校理念，全力打造“激情教育、高效课堂、多彩社团、德育新实、精细化管理、人工智能+教育”等六张新实特色教育名片，快速奔跑在教育的康庄大道上，正朝着建设一所市内一流省内知名的品牌学校阔步前进。 

## 1.2 办学规模与特色

学校秉持“德育立校，科研兴校，质量强校”办学方针，打造“激情校园、德育新实、多彩社团、高效课堂、精细化管理、智慧校园”六大特色品牌，坚持“五育并举”教育理念。建有校园宽带局域网、智能广播系统，教师队伍包含高级教师及党员教师
