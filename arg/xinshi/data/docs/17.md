# 2019-2020学年新实中学秋季拔河比赛

为丰富我校课外活动，加强各班之间的相互联系，促进学生德智体全面发展，增强学生体育锻炼的意识，增进同学之间的团结意识，以组织拔河比赛的形式为学生创造运动与交流的机会，提供展现个人、集体风采的舞台。培养学生集体荣誉感、组织观念、热爱集体、不怕吃苦、勇于拼搏和团队合作的精神，同时也是向学生进行德育教育的良好机会。

 

​    当比赛的哨声响起，只见参赛的队员们紧握长绳，卯足了劲拼命往后拉！当双方实力势均力敌，红色标记在两队之间左右移动，各队队员均使出九牛二虎之力，都想为自己班级争得荣誉。

本次比赛以班级为单位，各班代表队都挑选出了最强的比赛阵容，啦啦队早早聚集在比赛现场，个个摩拳擦掌，场面壮观,绳子绷紧的那一刻，双手攥紧的那一刻，哨声响起的那一刻，比赛开始啦！

 队员们暗自较劲，用力握紧绳子，铆足力了劲儿往后拉。场下观众大喊加油，为选手鼓劲助威。在一番激烈的角逐后，高三（1）班和高三（11）班进入总决赛，将比赛推入高潮

通过本次活动，充分增进了同学们的顽强拼搏精神，展现了良好的青春风采，也进一步增强了同学们的集体荣誉感和凝聚力，让同学们在乐趣中体验到了团队的含义。

比赛中，不论是哪只队伍，参赛同学都会全心倾力，顽强拼搏，加油呐喊的同学也是使出浑身解数，助威本班同学。选手的拼搏，观众的热情呐喊，使每场比赛都精彩万分。
