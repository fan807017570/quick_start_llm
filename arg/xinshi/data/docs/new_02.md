

# 关于表彰2016年高考质量先进单位的决定

 

## **关于表彰2016年高考质量先进单位的决定**

 

### 各县（市、区）教育局、各普通高（完）中：

 2015-2016学年度，我市各地、各普通高（完）中全面贯彻党和国家的教育方针政策，不断深化教育教学改革，为提高教育质量特别是高考质量作出了新的贡献。为鼓励先进，激励后进，市教育局决定对玉山县等13个优胜单位和弋阳县等9个进步单位予以表彰。

希望受表彰的单位戒骄戒躁，再接再厉，勇攀高峰，再创佳绩。各地、各普通高（完）中要向受表彰的单位学习，树立科学的质量观，努力拼搏进取，赶超先进，深化改革，强化管理，决战课堂，争取在新的一年里取得更大的成绩，为全面提高教育教学质量作出新的贡献。
